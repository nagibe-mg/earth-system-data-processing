***ESDP Homework 1***

This repository contains a Jupyter Notebook suitable to download data form the WCRP Coupled Model Intercomparison Project Phase 6 (CMIP6) datasets, more specifically, the geopotential height. 



The project includes simulations from more than 100 global climate models and around 45 institutions and organizations worldwide.
